How this works: This admin editor works locally. After editing, export items.json and replace data/items.json in the project before publishing. No login, no server, nothing is saved online.

Real-data readiness

This static admin validates image path format only — it cannot verify every file exists on disk. After exporting items.json, open the public catalog and product pages locally to confirm all images load.

Recommended replacement workflow

  1. Duplicate a sample product close to your item category.
  2. Change title, price, status, condition, description, and specs.
  3. Replace image paths with real image paths.
  4. Set featured only for important items.
  5. Export items.json.
  6. Replace data/items.json in the project.
